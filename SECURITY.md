# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a Vulnerability

Please do not open a public issue for security problems.

Report vulnerabilities by email to [ajimaru_gdr@pm.me](mailto:ajimaru_gdr@pm.me).

Include:

- a short description of the issue
- affected version(s)
- reproduction steps
- the impact you observed

## Scope

OctoPrint PandaBreath runs inside OctoPrint and relies on OctoPrint for authentication and access control. Relevant reports include:

- authentication or authorization bypasses
- arbitrary file or network access
- unsafe command execution
- information disclosure
- denial of service

Issues in OctoPrint itself should be reported to the OctoPrint project.
